# 專案名稱：線上租車平台

---
**專案介紹：**  
- 本系統以**HTML、CSS、PHP 與 MySQL**開發租車平台，並部署至**Google Cloud雲端伺服器(GCP)**，提供客戶租車功能與公司後台管理系統。
- 系統設計了完整的「前台客戶端」與「後台管理端」，，包含瀏覽車款、購物車系統、訂單管理以及會員管理功能。
- 此專案為小組專題作品，主要負責後端程式與資料庫設計，確保系統穩定運作並可支援多使用者同時操作。
- 改善租借流程，提升系統效能並改善使用者互動體驗。

---

| 類別 | 使用技術 |

## 功能介紹  
本系統分為兩個主要角色：一般用戶 (User) 與 管理員 (Admin)。

**前台用戶功能 (Client Side)**
會員系統：  
-會員註冊與登入 (Cookie 紀錄登入狀態)。
-忘記密碼 (透過 SMTP 發送 Email 找回密碼)。
-個人資料管理 (修改基本資料、上傳駕照圖片)。

租車服務：  
-產品展示：瀏覽所有可租車款、價格與剩餘數量。
-購物車系統：將車輛加入購物車、修改租借數量。
-線上預約：選擇租借開始時間，系統自動計算總金額並生成訂單。
-歷史訂單：查詢過往的租車紀錄與訂單狀態。

**後台管理功能 (Admin Side)**  
儀表板：整合客戶、帳戶、訂單與產品管理的入口。

產品管理：
-新增、修改、刪除車輛資訊 (包含上傳車輛照片、設定庫存與定價)。

訂單管理：
-檢視所有用戶訂單，租車時間時長、車款、價格和租還車狀態。
-可修改訂單狀態 (如：未取車、已還車)。

客戶管理：
-檢視所有會員資料，租車時間時長、車款、價格和租還車狀態。
-具備刪除違規用戶功能。
